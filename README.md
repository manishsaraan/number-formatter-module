Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install number-formatter-module`

## Usage

    var numFormatter = require('number-formatter-module');

    var formattedNum = numFormatter(30000);
  
  
  Output should be `30,000`
  ## Tests

  `npm test`
